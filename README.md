# carisurg-portfolio

A clinical data science portfolio completed as part of the CariSurg 
MedTech Pathways Programme (Cohort 2026), exploring the use of data 
analysis and artificial intelligence to support emergency department 
triage.

## Purpose

This repository is intended for clinical reviewers, healthcare 
professionals, and programme tutors interested in the application of 
data science and AI within emergency care. No programming knowledge 
is required to review the main outputs.

## Installation

No local installation required. All notebooks were developed in 
Google Colab. To run them, upload the notebook to 
[Google Colab](https://colab.research.google.com) and run each 
cell in order.

## Usage

Open any of the Week 0 notebooks in Google Colab:

- [Assignment 1 - Data Cleaning](notebooks/Assignment_1_DataCleaning.ipynb)
- [Assignment 2 - Advanced Cleaning](notebooks/Assignmnet_2__Advanced_Cleaning.ipynb)
- [Assignment 3 - Data Visualization](notebooks/Assignment_3_DataVisualization.ipynb)

The Week 1 research proposal is in `docs/`. The source dataset is 
not tracked in Git — see `data/README.md`.
